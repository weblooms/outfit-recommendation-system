## 🌟 Features
✅ Personalized outfit suggestions using OpenAI GPT API

✅ Filters by gender, occasion, season, and mood

✅ Google login for saved preferences

✅ Modern, funky UI with Tailwind CSS

✅ Mobile responsive

✅ Express.js backend with PostgreSQL DB
